Дьяк — портативная версия
=========================

Генератор кадровых документов с автоматическим русским склонением
ФИО и должностей по падежам.

Как пользоваться
----------------

1. Распакуйте всю папку в любое место (рабочий стол, флешка и т.п.).
2. Запустите dyak.exe — откроется окно «Дьяк».
3. На вкладке «Генерация» укажите:
   - таблицу с данными (xlsx),
   - шаблон документа (docx),
   - папку для готовых документов.
   Нажмите «Сгенерировать».
4. Готовые документы появятся в выбранной папке.

Установка Python и каких-либо зависимостей НЕ требуется — всё уже
внутри папки. Папку можно целиком перенести на другой компьютер.

Требования
----------

- Windows 10/11 (64-bit).

Справка по разметке шаблонов — на вкладке «Справка» в самой программе.
